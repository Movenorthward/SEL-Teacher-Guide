「用愛與技術，轉化每一個溝通瞬間」



SEL 導師暖心教戰手則 是一款專為教育工作者（導師、行政人員、校長）、學生及家長設計的 AI 溝通輔助工具。本系統結合了 SEL（社會情緒學習） 核心原則與 薩提爾（Satir）冰山理論，能幫助您在情緒高漲或面臨溝通衝突時，將內心的「真實情緒」轉化為「溫暖、專業且具備同理心」的有效對話。



🎯 三大核心功能



1\. 🔄 情緒轉化器 (SEL)



當您感到憤怒、無奈或急躁時，不要急著送出訊息！將您最原始的情緒話語輸入進來，AI 會根據雙方的角色關係，自動幫您轉化為得體、專業且聚焦於「解決問題」的回應。



支援多種角色切換：老師、家長、學生、行政人員、校長、教育局(處)長官、新聞媒體。



一鍵外語翻譯：內建「英文翻譯（簡單易懂版）」與「日文翻譯（專業敬語版）」，輕鬆應對外籍家長與學生。



2\. 🏔️ 薩提爾對話 (冰山理論)



別人或自己的一句話，往往只是「水面上的冰山一角」。



深層結構分析：輸入一句話，AI 將帶您看見水面下的 「行為、感受、觀點、期待、渴望」。



一致型溝通回應：系統會自動生成一段充滿好奇、接納與同理的薩提爾式回應，幫助建立真實的深度連結。



3\. 📖 SEL 教戰錦囊



內建社會情緒學習（SEL）的五大核心能力說明（自我覺察、自我管理、社會覺察、人際技巧、負責任的決定），隨時為您提供心理支持與指引。



📱 使用步驟指南



🔧 A. 如何使用「情緒轉化器」？



點擊畫面上方的 「情緒轉化器」 頁籤。



在左側面板選擇 「你的角色」 與 「對話對象」（例如：你是「老師」，對象是「家長」）。



點擊輸入框，直接打出您心裡最想抱怨或最原始的情緒話語（例如：「這孩子真的沒救了，家長也都不管！」）。



💡 提示：您也可以點擊「語音輸入」按鈕，直接對著麥克風說話（建議使用 Chrome 瀏覽器）。



點擊紅色的 「轉化暖心回應」 按鈕。



右側面板將出現經過 AI 潤飾的專業回應。您可以點擊 「複製建議文字」 直接貼到 Line 或 Email 中使用。



若需外語支援，可點擊下方的 「英文翻譯」 或 「日文翻譯」。



🔧 B. 如何使用「薩提爾對話」？



點擊畫面上方的 「薩提爾對話」 頁籤。



選擇 「你的身份」。



在輸入框中，輸入一句您當下最真實的感受或衝突對話（例如：「我真的不想管了」）。



點擊紫色的 「探索我的冰山」 按鈕。



右側將為您層層解析這句話背後的冰山結構，並給予您一段最適合的溝通起手式。



⚠️ 常見問題與注意事項



Q: 為什麼點擊轉化後，會出現「被 AI 底層安全機制攔截」的錯誤訊息？



A: 本系統串接 Google Gemini AI。為了符合國際 AI 安全規範，若您輸入的文字包含「極度嚴重的辱罵、仇恨言論或暴力字眼」，會被系統底層強制擋下。若遇到此情況，請稍微將最極端的字眼（如髒話）用符號 \*\*\* 替代，或稍微修飾後再次嘗試即可。



Q: 語音輸入沒有反應怎麼辦？



A: 語音辨識功能依賴瀏覽器內建的 Web Speech API。強烈建議使用 Google Chrome 或基於 Chromium 核心的瀏覽器（如 Edge），並確保您已允許網頁存取麥克風權限。



Q: 要如何部署或安裝到自己的學校使用？



A: 本專案為開源架構，程式碼中包含一個 apiKey 變數。資訊組長或開發者只需至 Google AI Studio 申請一組免費的 Gemini API Key，填入 src/App.jsx 檔案中，即可發佈到 Vercel 或 GitHub Pages 供全校使用。
